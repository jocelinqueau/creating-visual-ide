## Potential props

Editor tabs are a way to see the filename, it must be closeable, it must have a selected state that show the current selected file 

Tab can show if the file is saved or not

Tabs can add up in a scrobbable view or can stack up

Tabs can have an icon representing a possible language

Tabs can show an error state (like filename in red)

Tabs can also show a loading state for loading language system information for example
